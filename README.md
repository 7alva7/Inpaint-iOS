# [Inpaint-web](https://github.com/wudijimao/Inpaint-iOS)

A free and open-source inpainting app powered by coreml on iPhone / iPad / MacBook with M CPU.

基于 coreml 技术的免费开源 inpainting iOS App, OnDevice处理，无需服务器。
